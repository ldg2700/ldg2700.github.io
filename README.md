# Providelabs Website (Static Template)

This is a simple, responsive static site template for Providelabs Corporation.

## Pages

- `index.html` – Home (services, achievements, process)
- `about.html` – About (story, mission/vision/values)
- `services.html` – Service details
- `contact.html` – Contact details + email composer form

## How to use

1. Open `index.html` in a browser (double-click).
2. To deploy, upload the entire folder to any static host (Netlify, Vercel, GitHub Pages, cPanel, etc.).

## Customize

- Colors and layout: `assets/css/style.css`
- Mobile menu behavior: `assets/js/main.js`
- Images: `assets/images/`

## Contact defaults

The contact form composes an email to `luke.deleon@providelabscorp.com`.
If you prefer a different inbox (e.g., `info@...`), update it in `contact.html`.
